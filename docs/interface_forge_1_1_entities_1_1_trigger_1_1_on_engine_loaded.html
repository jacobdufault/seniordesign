Detailed Description

A trigger that is NOT deterministic. Instead, this allows for code to be executed when the game engine has been created (such as when a new level has started or a saved game has been loaded). This is primarily useful for dispatching custom events to notify the rendering engine of initial state.

Notice that this interface does not extend ISystem. This means that GlobalEntity, EntityIndex, TemplateIndex, etc, are not available for use. They may be null when OnEngineLoaded is called.
