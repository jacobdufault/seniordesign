enum Forge.Entities.GameSnapshotEntityRemoveResult

Represents the result of removing an entity.

Enumerator
Destroyed 

The entity was completely destroyed and no longer exists.

IntoRemoved 

The entity was moved into the Removed collection.

Failed 

The entity could not be removed (it was the singleton entity or was already in the Removed collection). Removing such an entity would cause invalid state in systems; the entity would not go through its life-cycle.

enum Forge.Entities.SystemExecutionOrdering

An enum that specifies the relative execution ordering of a group of systems. It expresses the execution ordering within the context of precisely two systems.

Enumerator
Concurrent 

The execution order doesn't matter; both systems can be ran concurrently.

BeforeOther 

This system should be executed before the other system.

AfterOther 

This system should be executed after the other system.
